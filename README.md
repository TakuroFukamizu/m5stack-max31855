# M5Stack + MAX31855 sample

## Requirements

### Hardware
- M5Stack Basic
- [K型熱電対温度センサモジュールキット(SPI接続)MAX31855使用 3.3V版](https://www.switch-science.com/catalog/864/)

### Software
- Arduino IDE
- M5Unified v0.0.5

## pin assign

| M5Stack | MAX3185 |
| ------- | ------- |
| 18      | SCK     |
| 19      | MISO    |
| 5       | SS      |
| 3.3V    | VCC     |
| GND     | GND     |
